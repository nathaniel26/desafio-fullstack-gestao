# Desafio Técnico: Sistema de Gestão Full Stack (C# + Angular)

Este repositório contém a solução completa para o desafio de gestão de usuários e produtos, desenvolvida conforme os requisitos técnicos solicitados: **.NET (C#)**, **Angular** e **SQL Server**.

A aplicação é composta por uma API REST robusta e uma interface de usuário moderna e reativa, garantindo uma experiência fluida com autenticação JWT e persistência de dados.

---

## 🏗️ Estrutura do Projeto

O projeto está dividido em dois diretórios principais.

### 🖥️ [Backend - API .NET](./backend/README.md)
* **Tecnologias:** .NET 8/9, Entity Framework Core, SQL Server.
* **Destaques:** Autenticação JWT, Migrations, Injeção de Dependência e Repositórios.
* **Configuração:** Connection String e comandos `dotnet ef`.

### 🎨 [Frontend - Angular App](./frontend/README.md)
* **Tecnologias:** Angular 18/19, Signals, Reactive Forms.
* **Destaques:** Interceptor de segurança, CRUD Reativo, Upload de Imagens com FormData.
* **Configuração:** Instalação de dependências `npm` e `ng serve`.
